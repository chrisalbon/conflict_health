=== Post Word Count ===Contributors: nickmomrikTags: posts, count, wordsStable tag: trunkCounts the total number of words in all posts or the number of words in a single post.== Installation ==1. Upload `post-word-count.php` to the `/wp-content/plugins/` directory1. Activate the plugin through the 'Plugins' menu in WordPress1. Place `<?php mdv_post_word_count(); ?>` in your templates.== Configuration ==The default is to count the total number of words in all posts. If you would like the number of words displayed for a particular post or page, use the following inside "the loop"`<?php mdv_post_word_count(true); ?>`
